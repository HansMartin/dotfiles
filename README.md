# Dotfiles


## Installation

Use the progs.csv file as source for packages, github repos, github .deb files (Idea stolen from Luke Smith)

__Format:__

```sh
./install.sh
```


This copies every file in the right place and installs all the specified packages

* WM: i3-gaps
* Terminal: st
* Editor: vim
* File Browser: vifm
* PDF Viewer: zathura
